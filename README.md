## Gendiff

Gendiff â€“ is a program that determines the difference between two data structures. This is a popular task for which there are many online services, such as http://www.jsondiff.com/. A similar mechanism is used when outputting tests or when automatically tracking changes in configuration files.

Gendiff features:

- Support for different input formats: yaml, json.
- Report generation in the form of plain text, stylish and json.

## Installation Guide 

```bash
git clone git@github.com:Serobabin/frontend-project-lvl2.git
npm ci
sudo npm link
```
